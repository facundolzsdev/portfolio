Mis Proyectos

Cotizador de Envíos

GEO-CARGA es una aplicación web que permite realizar cotizaciones de envíos dentro de Argentina. Muestra la distancia, el tiempo de entrega estimado, las instrucciones paso a paso de la ruta y un mapa interactivo, además de incluir validaciones y manejo de errores en tiempo real.

Stack: Spring Boot, WebClient, Axios, React-Bootstrap.

Northwind Traders

Sistema de ecommerce que utiliza una versión adaptada de la base de datos de la empresa ficticia Northwind Traders. Permite gestionar productos, clientes, empleados y pedidos, con control de acceso basado en roles (cliente, empleado y admin), así como la generación de reportes administrativos.

Stack: Jakarta EE, JSF + PrimeFaces (UI), MySQL, JPA/Hibernate.

Personal Event Planner API

API REST para gestionar eventos personales. Permite a cada usuario autenticado crear, editar, eliminar y consultar sus propios eventos (reuniones, recordatorios, etc.), mediante un sistema de autenticación y autorización JWT. Incluye integración con Swagger para una fácil exploración de los endpoints.

Stack: Spring Boot, Spring Security JWT, Spring Data JPA, MySQL.
